1. INSTALLATION
----------------

This release installs U.are.U Fingerprint Reader Driver (WBF) Version 5.0.0.5.

You must have administrator's right to install this software.

- You can select to run the setup program or save the setup program on your local disk.
- Follow the instructions on the installation.
- At the end of the installation you will need to reboot your computer.
- KB2685813 has to be installed first before attempting to install this driver package in Windows 7.


-----------------
2. COMPATIBILITY
-----------------

This driver is not supported by non-WBF legacy applications.

This Crossmatch U.are.U Fingerprint Reader Driver (WBF) is compatible with the following DigitalPersona/Crossmatch products:

	U.are.U 4500 Fingerprint Reader
	U.are.U 4000B Fingerprint Reader
	U.are.U 4500 Fingerprint Module
	U.are.U 4000B Fingerprint Module
	U.are.U Fingerprint Keyboard


-----------------------
3. SYSTEM REQUIREMENTS
-----------------------

Minimum system requirements:
- Pentium-class processor
- 45 MB disk space
- USB port
- One of the following Microsoft Windows operating systems (32-bit or 64-bit):
	Windows 10
	Windows 8.1
	Windows 8.0
	Windows 7
	

-----------------
4. RELEASE NOTES
-----------------

The included drivers - 
- Provide support for 32 and 64 bit operating systems
- FMA not included
- Add driver certification for Windows 10 (Gold Build 10240), Windows 10 (RS1 Build 14393), Windows 10 (RS2 Build 15063), Windows 10 (RS3 Build 16299)
